# Kakeikin Portfolio

Personal portfolio site built with Vite and React, published with GitHub Pages.

## Local Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy

This repository includes a GitHub Actions workflow in `.github/workflows/deploy.yml`.
After pushing to `main`, set GitHub Pages to use **GitHub Actions** as the source.

The live site will be:

```text
https://kakeikin.github.io
```

## Customize

Edit the profile text, project list, and contact links in:

```text
src/App.jsx
```
